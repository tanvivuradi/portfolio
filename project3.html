3D Design Preview of UBC Okanagan

Click and drag to rotate the model.

Developed a 3D model in Unity and exported it as a Unity package for the UBCO roadmap project, supporting both VR and AR applications. The applications allow users to place, manipulate, and persist models in the environment. The AR app, designed for mobile devices (Android/iOS), utilizes Google's Geospatial API for improved model localization and persistence using Street View. The VR app, built for Oculus, enables users to build and iterate on environments in a virtual setting, with the current version mimicking the area north of the UBCO campus.
